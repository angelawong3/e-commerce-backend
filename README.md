### Description

An app using Express.js API and Sequelize to interact with a MySQL database

### Installation

```
git clone git@github.com:angelawong3/e-commerce-backend.git
cd e-commerce-backend
npm install
mysql -u root
source db/schema.sql;
quit
npm run seed
```

### Usage

```
node server.js
or
npm run start
```
